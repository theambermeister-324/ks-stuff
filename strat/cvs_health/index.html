Scope of Work

In-Scope (Phase 1 PoC)

  • Component accessibility audit for 10–15 high-impact Digital Pulse components
  • Governance framework implementation with validation requirements before publishing
  • Integration with Deque/axe-core accessibility testing
  • Manual compliance reporting and executive scorecards via Professional Services
  • ROI analysis with metrics narrative aligned to CVS's existing compelling metrics
  • Stakeholder onboarding sessions

Out-of-Scope (Phase 1)

  • CI/CD pipeline enforcement automation
  • Real-time executive dashboards
  • AI-driven compliance monitoring
  • Cross-design-system consolidation
  • Figma data integration (dependent on access)
  • Full network/firewall integration
POC Sandbox Approach: The POC will operate as a "teaser that shows what's there" — demonstrating value without requiring full network integration. This addresses the firewall constraint while minimizing re-work risk.
