Program Overview

The PCRI Alpha Program is a structured validation initiative to test the Product Context Readiness Index assessment with a curated cohort of design system leaders and practitioners before public launch.

Terminology Note: Archetype vs. Journey

The PCRI framework distinguishes between two concepts:

  • Archetype = Where you are (maturity state based on PCRI score)
    • Builder (0-44): Establishing foundations
    • Adopter (45-54): Rapid enablement needed
    • Optimizer (55-74): Ready for transformation
    • Transformer (75-100): AI workflow expansion ready
  • Journey = Recommended investment path based on fit criteria
    • J1 Foundation: Build system of record first
    • J2 Acceleration: Fast activation for trigger-driven orgs
    • J3 Activation: Full transformation program
    • J4 Intelligence: MCP activation for mature orgs

Alpha reports include both: the archetype classification AND recommended journey path.

Alpha Process: Guided Delivery Model

Unlike the public release, the alpha program uses a human-in-the-loop approach for validation and calibration:

  1. Participant completes assessment — No PDF download available; results submitted for review
  2. Director of Delivery reviews submission — Validates scoring, calibrates archetype assignment, prepares personalized insights
  3. Curated PDF generated — Director adds contextual notes, industry comparisons, and tailored recommendations
  4. 1:1 debrief call scheduled — Director delivers PDF and walks through findings (30 min)
  5. Feedback captured — Participant provides input on accuracy and value

Alpha Objectives

  1. Validate question clarity — Identify confusing or ambiguous questions based on completion patterns and feedback
  2. Test archetype accuracy — Director of Delivery compares PCRI output to their assessment of participant's actual maturity
  3. Calibrate scoring algorithm — Adjust pillar weights and thresholds based on real-world validation
  4. Refine industry benchmarks — Collect baseline data across industries to power "vs. your industry" comparisons
  5. Optimize conversion funnel — Measure drop-off points and time-to-complete to improve UX
  6. Generate early pipeline — Identify qualified leads ready for IPE conversations

PCRI Lite vs. Full PCRI: The alpha assessment uses a simplified 10-question, ~10-minute format ("PCRI Lite") that maps to the full 5-pillar framework:

Full PCRI (5 Pillars) Weight Alpha Assessment Coverage
Culture 25% Captured via Director of Delivery calibration
Foundation 20% 3 questions (design system state, governance, documentation)
AI Readiness 20% 3 questions (AI tools, API maturity, product data)
Governance 20% 3 questions (executive sponsor, budget, change capacity)
Delivery 15% Captured via Director of Delivery calibration

The Director of Delivery supplements PCRI Lite scores with qualitative assessment of Culture and Delivery pillars during the debrief process. Post-alpha, the full 27-question PCRI will be available.

Debrief Call Feedback Questions

Director of Delivery should cover these questions during or immediately after each debrief call:

Required Questions (ask during debrief)

  1. Question clarity: Were any assessment questions confusing or hard to answer? Which ones?
  2. Time perception: Did the assessment feel too long, too short, or about right?
  3. Archetype accuracy: "Based on our discussion, does this archetype match where you see yourselves?" (Director validates)
  4. Report value: How useful is this curated report for your planning? (1-5 scale)

Director Calibration Notes (internal)

  1. Score validation: Does the PCRI score match your assessment of their maturity? If not, why?
  2. Archetype override: Should this participant be reclassified? Document reasoning.
  3. Question issues: Flag any questions where participant's answer didn't match reality.
  4. Industry benchmark: How does this compare to others in their industry?

Follow-up Survey (send post-call)

  1. How valuable was the debrief call? (1-5 scale)
  2. Would you recommend this assessment to a peer? (NPS: 0-10)
  3. Can we quote you in our marketing materials? (Yes / No)
  4. Any other feedback on the assessment or debrief experience?

  • Review question-level feedback; flag confusing questions
  • Compare PCRI scores to sales team's qualitative assessment of accounts
  • Calculate initial industry benchmarks

Week 4: Improvements

  • Rewrite/simplify low-clarity questions
  • Adjust pillar weights if correlation data suggests changes
  • Add/remove questions based on feedback
  • Improve results page based on value perception scores

Week 5: Public Launch Prep

  • Finalize production version of assessment
  • Create launch campaign assets (landing page, social, email)
  • Enable full Google Sheets integration for lead routing
  • Train sales team on interpreting PCRI scores
