# PCRI Alpha Program
## Director of Delivery Review Checklist

**Version:** 1.0  
**Last Updated:** 2026-02-16

---

## Overview

This checklist guides the Director of Delivery through reviewing each alpha participant's assessment submission before their 1:1 debrief call. The goal is to:

1. Validate the PCRI score and archetype assignment
2. Prepare personalized insights for the curated report
3. Capture calibration notes for algorithm improvement

---

## Pre-Review Checklist

### Before Opening the Submission

- [ ] Block 30-45 minutes for review + report curation
- [ ] Have the [PCRI Framework Whitepaper](PCRI_FRAMEWORK_WHITEPAPER.md) open for reference
- [ ] Have the [Industry Benchmarks](assessment_benchmark_calibration.md) open for comparison
- [ ] Open a new calibration notes document
