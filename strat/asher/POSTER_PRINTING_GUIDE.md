## Color Mode & File Format

### Current Status:
- ✅ **Format:** PNG (good for web, acceptable for print)
- ⚠️ **Color Mode:** RGB (web standard)

### For Professional Printing:

1. **CMYK Conversion:**
   - Most print services convert RGB → CMYK automatically
   - For color-critical work, convert in Photoshop/Illustrator first
   - Preview CMYK to check color shifts (especially greens/blues)

2. **File Format Options:**
   - **PNG:** Good, but large file sizes
   - **PDF:** Best for print (vector + raster, smaller file, CMYK support)
   - **TIFF:** Excellent quality, large files, CMYK support
   - **JPEG:** Not recommended (lossy compression)

### Recommended: Export as PDF
- Smaller file size
- Better CMYK support
- Print-ready
- Can embed fonts and vector elements

## Bleed & Safety Margins

### Standard Bleed:
- **Bleed:** 0.125" (1/8 inch) on all sides
- **Safety Margin:** 0.25" (1/4 inch) from edges for important text/graphics

### For 18" × 24" Poster:
- **Print Area:** 18" × 24"
- **With Bleed:** 18.25" × 24.25" (add 0.125" to each side)
- **Safe Area:** 17.5" × 23.5" (keep important content within this)

## File Preparation Checklist

### Before Sending to Printer:

- [ ] **Resolution:** At least 300 DPI at final print size
- [ ] **Dimensions:** Match exact poster size (e.g., 5400 × 7200 for 18" × 24" at 300 DPI)
- [ ] **Color Mode:** CMYK (or confirm printer accepts RGB)
- [ ] **File Format:** PDF preferred, PNG/TIFF acceptable
- [ ] **Bleed:** Add 0.125" bleed if design goes to edges
- [ ] **Safety Margins:** Keep text 0.25" from edges
- [ ] **File Size:** Under 50 MB (most services have limits)
- [ ] **Proof:** Review at 100% zoom for pixelation/artifacts
